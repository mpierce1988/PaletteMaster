# Palette Master

Palette Master is a .NET MAUI application designed to create and manage color palettes. It allows users to create color palettes, import them from Lospec, and quantize images to these palettes.
## Use Case

Palette Master is ideal for designers and developers who need to:

  Create and organize color palettes.
  Import color palettes from Lospec.
  Quantize images to the created color palettes.
  Export the quantized images to a file or folder.

## How to Run
### Prerequisites

  .NET MAUI installed on your development machine.

### Installation

  Clone the repository:    

    git clone https://github.com/mpierce1988/PaletteMaster.git
    cd src/PaletteMaster

  Open the solution in your preferred IDE (e.g., Visual Studio).

### Running the Application

  Set the PaletteMaster.Application project as the startup project.
  Run the application from your IDE.
